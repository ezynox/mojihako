かな入力パッド

ひらがな・カタカナ・濁音・小書き仮名・記号を、文字表からすばやく入力できます。

結果を別のツールへ送る 全角・半角 ひらがな・カタカナ 原稿用紙

かな入力パッドのGuide section

  1. ひらがな・カタカナ・濁音・小書き仮名・記号のタブから、入力したい文字を選びます。
  2. 文字をクリックまたはタップすると、入力結果欄のカーソル位置に挿入されます。
  3. [コピー] で入力結果をコピーし、フォームやメモ、SNS などに貼り付けられます。
  4. [⌫] で直前の 1 文字を削除し、[クリア] で入力結果をまとめて消去できます。
  5. 文字表では方向キーで移動し、Enter または Space で文字を入力できます。

対応文字

ひらがな・カタカナの基本文字に加えて、濁音・半濁音、 拗音や促音に使う小書き仮名、句点・読点・鉤括弧・長音符などの日本語記号を入力できます。

五十音レイアウト

五十音レイアウトをオンにすると、あ行を右端に置く伝統的な五十音順で表示します。 一覧から目で探したいときに便利です。

📱モバイルでは、縦向きでは通常の配列を使用し、横向きにすると五十音レイアウトに自動で切り替わります。

かな文字をすばやく入力・コピーする

かな入力パッドは、ひらがな・カタカナ・小書き仮名・日本語記号を文字表から選んで入力するためのツールです。 ふりがな、短い日本語フレーズ、作品名や商品名の一部など、 少しだけかな文字を入力したいときに使えます。

📌 こんなときに便利

場面 Guide section
小書き仮名を探したい ゃ、ゅ、ょ、っ、ァ などを一覧から選べます。
濁音・半濁音を確認したい が、ぱ、ガ、パ などをタブでまとめて確認できます。
日本語記号を入力したい 句点・読点・鉤括弧・長音符・三点リーダーなどをワンタップで入力できます。
入力した文字を別ツールへ回したい 入力結果を全角・半角変換、かな変換、原稿用紙ビューアへ送れます。

1. 収録している文字

基本的なひらがな・カタカナのほか、日本語入力でよく使う派生文字と記号をまとめています。

  • ひらがな・カタカナ： あ・い・う・え・お / ア・イ・ウ・エ・オ
  • 濁音・半濁音： が・ざ・だ・ば・ぱ / ガ・ザ・ダ・バ・パ 系列
  • 小書き仮名： ゃ・ゅ・ょ・っ / ャ・ュ・ョ・ッ / ァ・ィ・ゥ・ェ・ォ
  • 日本語記号： 。 、 「 」 『 』 ー 々 … など

2. 検索で文字を絞り込む

文字表から探すだけでなく、検索欄に読みやローマ字を入れて絞り込めます。

  • ka → か、カ
  • tsu → つ、ツ
  • xya → ゃ、ャ

小書き仮名や濁音・半濁音のように、一覧から探すと少し手間な文字を出したいときに便利です。

3. 日本語記号もまとめて入力

句点（。）・読点（、）・鉤括弧（「」）・二重鉤括弧（『』）・長音符（ー）などは、 日本語の文章や会話文でよく使う記号です。 記号タブから選ぶだけで、入力結果欄にそのまま挿入できます。

4. 他の MojiHako ツールと組み合わせる

入力した文字はコピーして使えるほか、他のツールへ送って追加処理できます。

  • 全角・半角変換：半角カタカナや文字幅を整えたいとき
  • ひらがな・カタカナ変換：入力後に文字種を切り替えたいとき
  • 原稿用紙：入力した文章を縦書き原稿用紙で確認したいとき

よくある質問

Q. 入力したテキストはサーバーに送信されますか？

送信されません。すべての処理はブラウザ内で完結します。

Q. スマートフォンでも使えますか？

はい。スマートフォンにも対応しています。 画面を横向きにすると、五十音レイアウトで使用できます。

Q. 半角カタカナは入力できますか？

かな入力パッドは通常のひらがな・全角カタカナを基準にしています。 ｶﾀｶﾅ のような半角カタカナが必要な場合は、全角・半角変換ツールを合わせてご利用ください。
