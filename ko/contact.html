문의

MojiHako를 사용하면서 불편한 점이 있거나 버그 제보, 개선 의견, 기능 제안, 기타 비즈니스 문의가 있으신 경우 아래 연락처로 연락해 주시기 바랍니다.

문의 가능한 내용

  • 도구가 정상적으로 작동하지 않는 경우
  • 변환 결과가 어색하거나 예상과 다른 경우
  • 추가되었으면 하는 일본어 텍스트 도구가 있는 경우
  • 오탈자, 잘못된 설명, 접근성 문제를 발견한 경우
  • 기타 사이트 이용과 관련된 문의

문의할 때 포함하면 좋은 정보

오류나 이상 동작을 제보할 때는 아래 정보를 함께 보내 주시면 확인에 도움이 됩니다.

  • 문제가 발생한 페이지 주소
  • 사용한 브라우저와 기기 종류
  • 입력한 예시 텍스트
  • 예상한 결과와 실제 결과
  • 가능하다면 화면 캡처
