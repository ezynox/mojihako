히라가나/가타카나 변환기

일본어 가나 문자를 히라가나 또는 가타카나로 변환합니다. 한자, 영문, 숫자는 그대로 유지됩니다.

변환 방식
결과를 다른 도구로 보내기 전각/반각 Kana Pad 원고지

한자의 읽는 법은 자동으로 추정하지 않습니다. 이 도구는 이미 입력된 히라가나와 가타카나만 변환합니다.

사용 방법

  1. 입력창에 변환할 텍스트를 붙여넣습니다.
  2. [히라가나 → 가타카나] 또는 [가타카나 → 히라가나] 중 원하는 방식을 선택합니다.
  3. 자동 변환이 켜져 있으면 결과가 실시간으로 나타납니다.
  4. 자동 변환을 끄면 [변환] 버튼으로 직접 변환할 수 있습니다.
  5. [복사] 버튼을 눌러 필요한 곳에 붙여넣으세요.

변환 예시

히라가나 → 가타카나

とうきょう → トウキョウ

가타카나 → 히라가나

カタカナ → かたかな

다른 문자는 유지

東京 ab123 → 東京 ab123

주의사항

! 이 도구는 한자의 일본어 읽는 법을 추정하지 않습니다. 예를 들어 '東京'을 'とうきょう'로 바꾸는 기능은 별도의 후리가나 변환 도구가 필요합니다.

히라가나·가타카나 변환 가이드

일본어 텍스트를 다루다 보면 히라가나와 가타카나를 맞바꾸어야 할 때가 있습니다. 단어 몇 개 정도는 직접 고치면 되지만, 내용이 길어지면 일일이 수정하기 번거롭습니다. 본 변환기는 입력된 가나 문자를 히라가나에서 가타카나로, 또는 가타카나에서 히라가나로 간편하게 바꿔줍니다. 문장 속에 섞여 있는 한자, 영문, 숫자, 문장 부호는 원래 형태 그대로 유지됩니다.

📌 히라가나와 가타카나의 차이점

문자 특징 주로 쓰이는 상황
히라가나
ひらがな		 둥글고 부드러운 형태 일본어 고유어, 조사, 동사·형용사 어미, 후리가나
가타카나
カタカナ		 각지고 직선적인 형태 외래어, 외국 인명·지명, 의성어·의태어, 강조 표현

히라가나와 가타카나는 글자 모양만 다를 뿐 발음은 기본적으로 같습니다. 예를 들어 あ와 ア, か와 カ는 같은 소리를 나타냅니다. 문맥과 단어의 성격에 따라 쓰임새는 달라지므로 필요에 맞춰 사용하시면 됩니다.

1. 히라가나를 가타카나로 변환하는 경우

외래어 표기를 정리하거나, 가나 표기 데이터의 형식을 하나로 통일한 때 유용합니다. 특히 일본 웹사이트는 회원 가입이나 티켓 예매를 할 때 이름의 발음을 적는 후리가나(혹은 요미가나) 칸에 가타카나만 입력하도록 제한하는 경우도 있습니다.

예를 들어 やまだ처럼 히라가나로 입력한 이름 표기를 ヤマダ처럼 가타카나로 바꿔야 할 때 [히라가나 → 가타카나] 변환을 사용합니다.

2. 가타카나를 히라가나로 변환하는 경우

가타카나로만 길게 적힌 단어나 문장은 한눈에 읽기 까다롭습니다. 일본어를 배울 때 가타카나 단어의 발음을 확인하고 싶거나, 만화·소설·게임 등에서 연출상 가타카나로만 적힌 대사를 익숙한 히라가나 형태로 편하게 읽고 싶을 때도 편리합니다.

예를 들어 カタカナ는 かたかな로, トウキョウ는 とうきょう로 변환할 수 있습니다. 다만 변환 결과가 실제 단어의 표준 표기나 자연스러운 표기를 의미하는 것은 아니므로 학습용으로 사용할 때는 문맥도 함께 확인하는 것이 좋습니다.

3. 이 변환기가 바꾸는 것과 바꾸지 않는 것

변환되는 것

  • 히라가나 → 가타카나: とうきょう → トウキョウ
  • 가타카나 → 히라가나: カタカナ → かたかな
  • 문장 속에 섞인 가나 문자만 선택적으로 변환

변환되지 않는 것

  • 한자: 東京
  • 영문: abc
  • 숫자: 123
  • 기호와 문장부호

주의(한자 독음 미지원): 본 변환기는 가나로 적힌 문자의 형태만 바꿉니다. 한자의 뜻이나 읽는 법을 추정해 東京을 とうきょう 또는 トウキョウ와 같이 바꾸는 기능은 지원하지 않습니다. 한자의 독음이 필요하다면 전용 후리가나 변환 도구를 이용해 주세요.

4. 히라가나·가타카나 변환이 필요한 상황

  • 일본어 학습 중 가나 표기를 비교하고 싶을 때
  • 이름의 후리가나를 히라가나 또는 가타카나 형식으로 맞춰야 할 때
  • 복사한 일본어 텍스트 안의 가나 표기를 한쪽으로 통일하고 싶을 때
  • 게임, 만화, 소설의 대사를 가타카나 표기로 통일하거나 그 반대로 바꾸고 싶을 때
  • 문서 편집이나 데이터 정리 과정에서 가나 표기를 일괄 변환해야 할 때

5. 반각 가타카나는 어떻게 변환하나요?

ｶﾀｶﾅ처럼 폭이 좁은 반각 가타카나는 일반적인 전각 가타카나 カタカナ와 다른 문자입니다. 이 페이지의 가나 변환기는 일반 히라가나와 전각 가타카나 간의 변환을 담당합니다. 반각 가타카나를 전각 가타카나로 바꾸거나, 전각 가타카나를 반각 가타카나로 바꾸고 싶으시면 전각/반각 변환기를 사용하세요.

🛑 변환 전 체크리스트

  • 히라가나로 바꿀지, 가타카나로 바꿀지 변환 방향을 확인하세요.
  • 한자가 포함된 텍스트는 가나 부분만 변환된다는 점을 확인하세요.
  • 반각 가타카나 변환이 필요하다면 전각/반각 변환기를 사용하세요.
  • 이름, 주소, 예약 정보처럼 중요한 입력값은 결과를 다시 확인하세요.

자주 묻는 질문

Q. 입력한 텍스트가 서버로 전송되나요?

아니요. 입력한 텍스트는 사용자의 브라우저 안에서만 처리되며 서버로 전송되지 않습니다.

Q. 한자를 히라가나나 가타카나로 바꿀 수 있나요?

한자의 읽는 법은 추정하지 않습니다. 예를 들어 東京은 그대로 유지되며, とうきょう이나 トウキョウ로 자동 변환되지 않습니다.

Q. 긴 글이나 대량의 문장도 한 번에 변환할 수 있나요?

입력창에 붙여넣을 수 있는 분량이라면 한 번에 변환할 수 있습니다. 긴 문장에서도 히라가나와 가타카나만 바뀌고, 한자와 영숫자는 그대로 유지됩니다.
