전각/반각 변환기

일본 웹사이트의 입력 양식이나 주소, 이름, 상품명 등을 정리할 때 자주 필요한 전각 문자와 반각 문자를 빠르게 변환합니다.

변환 방식
결과를 다른 도구로 보내기 히라가나/가타카나 Kana Pad 원고지

! NFKC 정규화는 ①, Ⅳ 같은 호환 문자까지 함께 변환하므로, 원문 형태를 그대로 보존해야 할 때는 변환 결과를 꼭 확인하세요.

사용 방법

  • 입력창에 변환할 텍스트를 붙여넣습니다.
  • 원하는 변환 방식을 선택합니다.
  • 자동 변환이 켜져 있으면 결과가 실시간으로 나타납니다.
  • 자동 변환을 끄면 [변환] 버튼으로 직접 변환할 수 있습니다.
  • [복사] 버튼을 눌러 필요한 곳에 붙여넣으세요.

전각과 반각의 차이

전각 문자는 정사각형 모양으로 한 글자 폭이 넓은 문자이며, 반각 문자는 일반 영문이나 숫자처럼 폭이 절반으로 좁은 문자입니다.

일본 사이트는 이름, 주소, 전화번호, 상품명을 입력할 때 전각과 반각 구분이 필요하기도 합니다.

NFKC 정규화란

NFKC 정규화는 컴퓨터가 다른 글자로 인식하는 복잡한 특수문자(①, Ⅳ)나 전각 문자(Ａ, １)를 표준적인 일반 텍스트 (1, IV, A, 1)로 통일해 주는 기능입니다.

텍스트에 섞여 있는 입력 오류나 불필요한 기호들을 표준 유니코드로 한 번에 정돈해 주기 때문에, 웹사이트 검색이나 데이터 분석을 할 때 정확도를 높여줍니다.

일본 사이트 입력 오류 해결을 위한 전각·반각 변환 가이드

일본 웹사이트에서 회원가입을 하거나 아마존·메루카리 직구, 호텔 예약 등을 진행할 때 “전각으로 입력해 주세요(全角で入力してください)” 또는 “반각으로 입력해 주세요(半角で入力してください)”라는 오류 메시지 때문에 막히는 경우가 많습니다. 이는 전각과 반각, 즉 문자 폭을 구분해서 처리하는 입력 양식 때문입니다. 아래 안내에 따라 오류 원인을 확인하고 필요한 형식으로 간편하게 변환해 보세요.

1. 전각 문자와 반각 문자의 뜻

  • 전각 문자(全角文字, full-width characters):정사각형(1:1 비율) 모양으로, 한 글자가 온전한 한 칸 폭을 차지
  • 반각 문자(半角文字, half-width characters):전각의 절반(1:0.5 비율)으로 일반적인 영문이나 숫자처럼 차지하는 폭이 좁음

❓ 일본 직구 사이트에서 전각 오류가 왜 생기나요?

겉보기에는 비슷해 보이는 알파벳 A와 Ａ, 숫자 1과 １, 가타카나 ｶ와 カ를 컴퓨터 시스템은 서로 다른 문자로 인식합니다. 주소를 복사해서 붙여넣거나, 다른 앱에서 텍스트를 가져오거나, 스마트폰의 자동 완성 기능을 쓰면 전각과 반각이 의도치 않게 섞여 들어가 오류가 발생하곤 합니다.

2. 가타카나 후리가나 입력: 전각 vs 반각

일본 웹사이트의 이름 후리가나 입력란은 전각 가타카나를 요구하는 경우가 많습니다. 만약 여권 영문 이름을 가타카나로 변환하여 입력하였는데 오류가 난다면, 문자 폭이 좁은 반각 가타카나로 입력되었기 때문일 수 있습니다. 이럴 때는 본 페이지의 [반각 가타카나 → 전각 가타카나] 기능을 이용하여 변환 가능합니다.

※단, 일부 오래된 예약 시스템이나 배송 시스템에서는 반각 가타카나(半角カナ)를 요구하는 경우도 있으니, 오류 안내 메시지를 꼭 확인해 주세요.

3. 전각 숫자와 반각 숫자가 다른 점

전각 숫자는 １２３４처럼 정사각형(1:1 비율)에 가까운 폭을 차지하며, 반각 숫자는 일반적인 영문 숫자 폭인 1234입니다. 결제 카드 번호, 우편번호, 전화번호 칸에 전각 숫자가 들어가면 오류가 발생하기도 하므로 [전각 ASCII → 반각 ASCII]를 통해 숫자를 반각으로 정리할 수 있습니다.

4. 눈에 잘 보이지 않는 전각 공백 문제

찾기 까다로운 입력 오류 원인 중 하나는 공백입니다. 일본어 입력기가 켜진 상태거나 윈도우에서 Alt + =로 전각 입력 모드가 켜진 상태로 스페이스를 눌러 띄어쓰기를 입력하면 일반 공백이 아니라 넓은 전각 공백(　)이 입력될 수 있습니다. 성과 이름 사이, 주소 중간, 건물명 앞뒤에 전각 공백이 포함되어 있으면 사이트가 입력값을 제대로 인식하지 못할 수 있습니다. 오류 원인을 찾기 어렵다면 [전각 공백 → 반각 공백] 변환을 실행해 보시기 바랍니다.

5. 대량의 데이터 정리는 'NFKC 정규화'로 한 번에

일본 상품명을 대량으로 수집해 데이터 분석을 하거나, 쇼핑몰·마켓 등록용 텍스트를 정리할 때는 NFKC 정규화 기능이 유용합니다. 텍스트 내에 임의로 섞여 있는 호환 문자(①, Ⅳ, ㍿ 등)와 전각 영숫자를 더 표준적인 일반 텍스트(1, IV, 株式会社, 일반 영숫자)로 통일하여 검색이나 비교가 쉬워집니다.
