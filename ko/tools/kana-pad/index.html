Kana Pad(일본어 입력기)

일본어 키보드나 IME가 없어도 히라가나와 가타카나, 문장 기호 등을 바로 입력합니다.

결과를 다른 도구로 보내기 전각/반각 히라가나/가타카나 원고지

Kana Pad란

Kana Pad는 일본어 키보드나 IME 없이 히라가나, 가타카나, 기호 등을 온라인에서 바로 입력할 수 있는 도구입니다.

모바일에서 일본어를 자주 입력한다면 Kana Pad를 즐겨찾기 하고 사용해 보세요!

Kana Pad 사용 방법

  1. [あ 히라가나], [が 히라가나 탁음·반탁음], [ア 가타카나], [ガ 가타카나 탁음·반탁음], [ゃ 작은 가나], [「」문장 기호] 탭에서 필요한 문자를 선택합니다.
  2. 선택한 문자는 입력 결과창 커서 위치에 삽입되며 [복사] 버튼으로 전체 문장을 복사할 수 있습니다.
  3. [삭제] 버튼으로 입력 결과를 전부 삭제합니다. [⌫] 버튼으로 이전 한 글자를 삭제합니다.
  4. 문자표에서는 방향키로 이동할 수도 있고 Enter 또는 Space로 문자를 입력할 수 있습니다.
  5. 키보드만 사용할 때는 Tab과 Shift+Tab을 이용해 입력 영역, 작업 버튼, 탭, 문자표 사이를 이동할 수도 있습니다.

지원 문자

오십음도 기반의 히라가나와 가타카나뿐만 아니라 히라가나·가타카나 탁음과 반탁음(が・ざ・だ・ば・ぱ / ガ・ザ・ダ・バ・パ 계열), 요음・촉음(ゃ・ゅ・ょ・っ), 작은 가나(ァ・ィ・ゥ・ェ・ォ), 그리고 일본어식 문장 부호(쿠텐과 토텐, 따옴표와 괄호, 장음, 반복 부호, 말줄임표 등)을 제공합니다.

오십음도 배열

선택하면 입력 패드가 전통 세로쓰기 형태의 오십음도 순서로 표시됩니다. 모음은 위에서 아래로 あ・い・う・え・お 순서로 배치되고, 열은 오른쪽에서 왼쪽으로 진행되며 あ행이 가장 오른쪽에 놓입니다.

📱모바일에서는 세로 화면일 때 일반 배열을 사용하고, 가로 화면으로 돌리면 오십음도 배열로 자동 전환됩니다.

검색 예시

로마자

ka → か, カ

가나

つ → つ, っ, ツ, ッ

한글

쿠 → く, ク, ぐ, グ

일본어 키보드 없이 히라가나·가타카나 입력하기

일본어를 입력해야 하는데 일본어 키보드나 IME 입력기가 설정되어 있지 않을 때가 있습니다. 직장 PC나 공용 컴퓨터처럼 언어 설정을 마음대로 변경하기 어려운 환경이거나 일본어를 가끔씩만 사용할 뿐이라면 매번 일본어 자판을 추가하는 과정이 부담스러울 수도 있습니다. Kana Pad는 별도의 설치 없이 브라우저에서 바로 히라가나, 가타카나 및 문장 기호를 입력할 수 있는 웹 도구입니다.

📌 Kana Pad가 필요한 상황

상황 활용 방법
일본어 키보드·IME가 없는 기기 설정 변경 없이 브라우저에서 필요한 가나 문자를 입력합니다.
일본 직구·예약 사이트 이용 상품명, 이름, 후리가나, 간단한 일본어 문구를 빠르게 입력합니다.
모바일에서 간단히 입력 앱 설치 없이 브라우저에서 열어 필요한 문자를 선택합니다.
일본어 기호 입력 장음 기호, 일본어 쉼표·마침표, 괄호, 반복 부호 등을 쉽게 찾습니다.

1. 일본어 자판 설정 없이 가나 입력하기

일본어를 자주 사용하지 않는다면 IME 설정이나 입력 방식을 익히는 과정이 번거로울 수 있습니다. Kana Pad는 화면에 표시된 오십음도 문자표에서 필요한 글자를 직접 선택해 입력하는 방식입니다. 자판 배열을 외우지 않아도 히라가나와 가타카나를 눈으로 확인하면서 입력이 가능합니다.

예를 들어 일본어 단어를 검색하거나 사이트의 입력란에 짧은 가나 문구를 넣어야 한다면 필요한 문자를 클릭이나 터치를 통해 조합한 뒤 복사하여 붙여넣을 수 있습니다.

2. Kana Pad에서 입력할 수 있는 문자

Kana Pad는 기본 히라가나와 가타카나뿐 아니라 일본어 입력에서 자주 필요한 기호도 함께 제공합니다. 일반 키보드에서 찾기 어려운 문자도 탭을 전환해 바로 선택할 수 있습니다.

  • 기본 히라가나·가타카나: あ・い・う・え・お / ア・イ・ウ・エ・オ
  • 탁음·반탁음: 히라가나 が・ざ・だ・ば・ぱ 계열, 가타카나 ガ・ザ・ダ・バ・パ 계열
  • 요음·촉음: ゃ・ゅ・ょ・っ / ャ・ュ・ョ・ッ
  • 일본어 문장 부호: 。 、 「 」 『 』 ー 々 … 등

3. 검색 기능으로 빠르게 문자 찾기

오십음도 표에서 원하는 글자를 찾기가 어렵다면 검색 기능을 사용해 보세요. 로마자, 가나, 한글 발음으로 직접 검색해 관련 문자를 찾을 수 있습니다.

  • ka 검색 → か, カ
  • つ 검색 → つ, っ, ツ, ッ
  • 쿠 검색 → く, ク, ぐ, グ

특히 작은 가나, 탁음, 반탁음처럼 헷갈리기 쉬운 문자를 찾을 때 유용합니다.

4. 일본어 특수 기호와 문장 부호 입력

일본어 문장에서는 보통 일반적인 쉼표와 마침표 대신 、와 。를 사용합니다. 대사나 인용에는 「 」, 『 』 같은 괄호가 쓰이고, 외래어 표기에서는 장음 기호 ー가 자주 등장합니다.

특수 문자 찾기가 까다롭다면 Kana Pad에서 필요한 기호를 선택해 삽입하세요.

5. 입력한 텍스트를 다른 도구로 이어서 사용하기

작성한 텍스트는 복사해서 그대로 붙여넣기 할 수 있고, 필요하다면 MojiHako의 다른 도구로 보내 추가로 작업합니다.

  • 전각/반각 변환: 반각 가타카나나 전각 문자 형식을 조정해야 할 때
  • 히라가나/가타카나 변환: 입력한 가나 문자의 종류를 바꾸고 싶을 때
  • 원고지: 입력한 일본어 문장을 원고지 형식으로 확인하고 싶을 때

6. 키보드만으로 사용하는 방법

마우스를 사용하지 않고 키보드만으로도 주요 기능을 조작할 수 있습니다. 입력 영역, 버튼, 문자표 사이를 이동하면서 원하는 문자를 선택해 입력 가능합니다.

  • 십자 방향키: 문자표 안에서 선택 위치 이동
  • Enter 또는 Space: 선택한 문자 입력
  • Tab / Shift+Tab: 입력창, 버튼, 탭, 문자표 사이 이동

자주 묻는 질문

Q. 입력한 텍스트가 서버로 전송되나요?

아니요. 입력한 텍스트는 사용자의 브라우저 안에서만 처리되며 서버로 전송되지 않습니다.

Q. 스마트폰에서도 사용할 수 있나요?

네. 모바일에서도 사용할 수 있습니다. 자주 사용한다면 즐겨찾기나 홈 화면 바로가기로 등록해 보세요.

Q. 히라가나로 입력한 뒤 가타카나로 바꿀 수 있나요?

네. 입력한 텍스트를 복사한 뒤 히라가나/가타카나 변환 도구로 보내 변환합니다. 반대로 가타카나를 히라가나로 바꾸는 것도 가능합니다.

Q. 반각 가타카나도 입력할 수 있나요?

Kana Pad는 기본적으로 일반 히라가나와 전각 가타카나를 기준으로 합니다. ｶﾀｶﾅ처럼 폭이 좁은 반각 가타카나가 필요하다면 전각/반각 변환 도구를 함께 사용하세요.
